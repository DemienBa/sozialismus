## Die Struktur

### 15 Achsen in 5 Feldern

**STRATEGIE (A)**
- A1: Reform ↔ Revolution
- A2: Arbeiterklasse ↔ Alle Unterdrückten
- A3: Staat nutzen ↔ Staat überwinden

**ÖKONOMIE (B)**
- B1: Markt ↔ Plan
- B2: Staatseigentum ↔ Selbstverwaltung
- B3: Arbeit befreien ↔ Arbeit reduzieren

**GESELLSCHAFT (C)**
- C1: Klasse zuerst ↔ Intersektional
- C2: Natur als Ressource ↔ Rechte der Natur
- C3: National ↔ International

**KULTUR (D)**
- D1: Utopie im Kampf ↔ Utopie ausmalen
- D2: Institutionen ↔ Transformation
- D3: Säkular ↔ Spirituell offen

**PRAXIS (E)**
- E1: Klasse vereinen ↔ Bewegungen vernetzen
- E2: Bundespolitik ↔ Lokal + Global
- E3: Theorie ↔ Erfahrung

## Weiße Flecken: Untertheoritisierte Kombinationen

### 1. SPIRITUELLER RÄTEKOMMUNISMUS
```
D3=5 (Spirituell) + B2=5 (Selbstverwaltung) + A3=5 (Anti-Staat)
```
**Frage:** Kann es eine spirituelle Arbeiterselbstverwaltung geben?
**Ansätze:**
- Befreiungstheologie + Räte?
- Quäker-Kommunen?
- Mystischer Anarchismus (Gustav Landauer?)

**Literatur (spärlich):**
- Gustav Landauer "Aufruf zum Sozialismus" (1911) – enthält mystische Elemente
- Martin Buber "Pfade in Utopia" – kommunitärer Sozialismus mit spiritueller Dimension

**Lücke:** Kaum systematische Verbindung von Rätekonzept und Spiritualität

---

### 2. ETATISTISCHER ÖKOSOZIALISMUS
```
A3=1-2 (Staat nutzen) + C2=5 (Rechte der Natur)
```
**Frage:** Kann ein starker Staat Rechte der Natur durchsetzen?
**Ansätze:**
- Chinesischer "Öko-Leninismus"?
- Staatlicher Degrowth?

**Literatur (sehr spärlich):**
- Diskussionen über "War Communism" für Klimakrise (Andreas Malm)
- Aber: Malm bleibt vage zur Staatsform

**Lücke:** Wie würde eine demokratisch-leninistische Ökodiktatur aussehen?

---

### 3. REFORMISTISCHER ANARCHISMUS
```
A1=1 (Reform) + A3=5 (Staat überwinden)
```
**Frage:** Kann man den Staat schrittweise erodieren?
**Problem:** Logisch fast widersprüchlich

**Ansätze:**
- Präfigurative Politik als "Reform von unten"?
- Duale Macht langsam aufbauen?

**Literatur:**
- Colin Ward "Anarchism in Action" – Anarchismus im Alltag, ohne Revolution
- Erik Olin Wright "Reale Utopien" – Erosion statt Revolution

**Lücke:** Keine systematische Theorie des gradualistischen Anarchismus

---

### 4. POST-WORK-LENINISMUS
```
B3=5 (Weniger Arbeit) + A3=2 (Staat nutzen) + E3=1 (Avantgarde)
```
**Frage:** Kann eine Avantgarde-Partei Post-Work durchsetzen?
**Ansätze:**
- Automatisierung durch Staatsplanung?
- UBI von oben?

**Literatur:**
- Aaron Bastani "Fully Automated Luxury Communism" – berührt das Thema
- Aber: Unscharf zur Staatsform und Strategie

**Lücke:** Keine leninistische Post-Work-Theorie

---

### 5. NATIONALER ÖKOSOZIALISMUS
```
C3=1 (National) + C2=5 (Natur) + B3=5 (Degrowth)
```
**Frage:** Kann es Degrowth nur in einem Land geben?
**Problem:** Klimakrise ist global, Degrowth-Nationalismus wäre regressiv

**Literatur:**
- Fast nichts – aus gutem Grund?
- Gefahr: Rechtspopulistische Ökologie ("Umweltschutz = Heimatschutz")

**Lücke:** Bewusst leer? Oder gibt es progressive Varianten?

---

### 6. KLASSENREDUKTIONISTISCHER FEMINISMUS
```
C1=1 (Klasse zuerst) + gleichzeitig starker Feminismus-Fokus
```
**Frage:** Wie argumentierten Marxist*innen, die Feminismus ernst nahmen aber Klasse priorisierten?
**Historische Figuren:**
- Clara Zetkin
- Alexandra Kollontai (teilweise)

**Literatur:**
- Gut dokumentiert für historische Periode
- Heute: Wenige vertreten das noch explizit

**Status:** Nicht wirklich "weiß" – eher historisch verortet

## Logisch Schwierige Kombinationen

### Widerspruchspaare

| Kombination | Warum schwierig? |
|-------------|------------------|
| A3=5 + B1=5 | Wie zentral planen ohne Staat? |
| A1=1 + D2=5 | Permanente Transformation durch Reform? |
| E3=1 + A2=5 | Avantgarde für alle Unterdrückten? |
| B2=5 + C3=1 | Nationale Selbstverwaltung? |

### Aber: Widersprüche können produktiv sein!

- **Zapatistas:** A3≈5 aber mit Elementen staatlicher Anerkennung
- **Rojava:** B2≈5 + C1≈5 (radikal feministisch) in Kriegszustand
- **Mondragon:** B2≈5 in kapitalistischem Markt
